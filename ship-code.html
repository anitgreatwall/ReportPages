🎯 高级发布策略 — 降低上线风险，快速验证与回滚
🔀 Feature Toggle
功能开关，通过配置动态开启或关闭某个功能，无需重新部署，快速回滚
🐦 Canary Deploy
金丝雀发布，先将新版本灰度给小比例用户（如 5%），观察稳定后逐步扩量至全量
🧪 A/B Test
将用户随机分成 A/B 两组，分别体验不同版本，通过数据对比决策最优方案
